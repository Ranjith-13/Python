This is a small exploratory data analysis and data visualization project using data from Spotify using Python.
Conducted data cleaning to perform data analysis and data visualization of the Spotify dataset using Python.(Pandas, NumPy, Matplotlib and Seaborn)

Spotify Data Analysis makes use of secondary data from Spotify and produces data about music and provides valuable insights.

Some of the Objectives derived using this project:
1)Top 10 most popular songs on Spotify
2)Top 10 least popular songs on Spotify
3)Top 5 Genres by Popularity
and some more,.
